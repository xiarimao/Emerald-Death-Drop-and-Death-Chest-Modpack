# Emerald-Death-Drop-and-Death-Chest-Modpack
本模组由AI辅助开发。旨在为你的游戏增添更丰富的死亡惩罚与战利品探索体验。  玩家死亡时，会在原地生成一个专属战利品箱子，内含原本会掉落的所有物品（除按比例保留的绿宝石外） 绿宝石可按百分比掉落，剩余部分会在你复活后自动返还背包 箱子上方自动放置告示牌，显示死亡玩家的名字，方便识别 箱内物品带有搜索时间标签，兼容 Search Carefully Mod 的搜索动画 箱子可配置在指定时间后自动消失（默认5分钟），无论是否被打开，避免世界被箱子堆满 所有功能均可通过配置文件灵活调整，满足不同服务器或单机需求
